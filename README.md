# Tomato Leaf Disease Detection using Deep Learning

This project detects tomato leaf diseases using CNN models.

## Models Compared
- MobileNetV2
- ResNet50
- EfficientNetB0 (Proposed Model)

## Dataset
Tomato Leaf Disease Dataset

## Results

| Model | Accuracy |
|------|------|
| MobileNetV2 | 92.3% |
| ResNet50 | 93.5% |
| EfficientNetB0 | 95% |

- TensorFlow
